# @steijnveer/create-file-based-router

Create a new file-based-router project with one command.

## Usage

```bash
# npm
npm create @steijnveer/file-based-router my-app

# yarn
yarn create @steijnveer/file-based-router my-app

# pnpm
pnpm create @steijnveer/file-based-router my-app

# bun
bun create @steijnveer/file-based-router my-app
```

Or create in the current directory:

```bash
npm create @steijnveer/file-based-router .
```
